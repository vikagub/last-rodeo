# Клуб Романтики: Последнее родео

Локальный прототип веб-игры в формате визуальной новеллы.

## Как открыть

Открой `index.html` в браузере.

## Где что лежит

- `index.html` — базовая разметка игрового экрана.
- `styles.css` — визуальный стиль, фон, диалоговое окно, кнопки, персонажи.
- `game.js` — сценарий и логика переходов между сценами.
- `assets/` — изображения персонажей.

## Как добавлять сцены

В `game.js` добавляй новый объект в `scenes`:

```js
sceneId: {
  background: backgrounds.stable,
  characters: [{ id: "polina", position: "left" }],
  speaker: "Поли",
  text: "Текст сцены",
  choices: [{ label: "Кнопка", next: "nextSceneId" }],
}
```

Кнопка `next` должна вести на существующий `sceneId`.
